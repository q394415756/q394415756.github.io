终端网络监控工具GlassWire

有一个客户场景需要对比两款软件对于网络带宽的占用情况，由于客户环境没有带宽监控类软件，所以就从终端侧监控软件入手。
终端监控软件GlassWire，可以实时监控网络流量，包括上传和下载。

功能

1.可视化网络监控

使用 GlassWire 简单易用的界面，在图表上查看您过去和现在的所有网络活动。 单击图表可查看哪些应用程序启动了传入或传出带宽，并立即查看应用程序正在与哪些主机通信。 主机会自动解析，并且还包括其原籍国/地区。单击“应用程序”和“流量”选项，按应用程序细分网络活动 和流量类型。

2.互联网安全

GlassWire 通过在易于理解的图表中可视化所有过去和现在的网络数据，为您的计算机或服务器增加了额外的互联网安全性。 立即查看通过 Internet 进行通信的每个应用程序或进程，然后更深入地了解您的计算机正在与谁或与什么进行通信。 每当计算机上的任何新内容访问网络时，GlassWire 都会通过谨慎的警报通知您，不会中断您的工作流程或服务器。

GlassWire 还有一个网络安全检查工具箱，例如系统文件更改检测、设备列表更改检测、应用程序信息更改检测、 ARP 欺骗监控，我们才刚刚开始。如果您的计算机或服务器正在与已知 IP 通信，我们的软件还会提醒您，或者 域威胁。我们正在努力在未来向 GlassWire 添加更多入侵检测规则。

不幸的是，全面的恶意软件检测和完美的网络入侵检测并不存在。我们的软件并不完美，但我们希望 GlassWire 将易于使用的网络可视化、防火墙管理和网络监控规则相结合，将在永无止境的战争中发挥重要作用。 间谍软件和恶意软件。GlassWire 的 Internet Security 是第二道防线，应始终与单独的主要防病毒产品一起使用。

3.互联网隐私保护

您的网络感觉慢吗？也许您的邻居已登录您的 WiFi 并观看 Netflix。查看网络上的设备列表 并在设备加入或离开时通过我们新的网络设备列表功能收到警报。

4.网络时光机&谨慎警报

GlassWire 是一款易于使用的工具，可以查看您过去和现在的网络活动。使用滑块回到过去并分析过去的网络 图表上的活动。使用已解析的主机按天、周和月详细检查带宽使用情况。

转到“流量监视器”选项卡或“日志分析”选项卡的图形视图，以准确查看应用首次访问 Internet 的时间、与哪些主机通信以及任何可用的 geoip 国家信息用一面小旗表示。查看图表，了解过去的安全警报以及详细的主机信息。 使用 GlassWire 可以很容易地在简单易懂的图表上查看过去的网络活动以及详细的网络活动。

我们专门设计了 GlassWire 警报系统，因此它不会打扰用户。重要的是要了解您的网络正在发生哪些变化 计算机或服务器，但需要关闭的不断弹出窗口令人沮丧。如果您正在制作截屏视频或根本不想要任何警报 period 转到 GlassWire 菜单并选择“暂停”以停止接收警报 24 小时。我们理解，在某些情况下，来自软件的警报可能是 很烦人。GlassWire 警报会短暂出现，然后消失在后台。您可以随时转到“日志分析”选项卡或查看图表 稍后详细查看警报。如果您根本不喜欢警报，可以在设置中禁用我们的安全警报，或者直接禁用我们的警报 在主设置窗口中。如果安全警报不是您的事，您可以将其全部禁用，只需将 GlassWire 用作网络监视器，但我们不会 推荐这个。

5.带宽使用监视器

使用 GlassWire 可以轻松跟踪您的每日、每周或每月带宽使用情况。转到“流量监控”选项卡的“使用情况表”视图，查看内容 应用、流量或主机使用的带宽最多。看到奇怪的东西了吗？单击应用程序或流量类型的图标，查看有关您的位置的更多详细信息 带宽与任何可用的 GeoIP 国家 IP 信息一起使用。通过将数据导出为 CSV 来独立分析流量。 请记住，GlassWire 仅跟踪安装它的计算机和服务器的带宽，而不会跟踪网络上的总带宽使用情况。

6.多远程服务器监控

您是否监控商业、游戏、VOIP 应用程序或 Web 应用程序的远程服务器？GlassWire 可轻松安装在服务器上 因此，您可以通过我们的远程访问功能在本地计算机上监控他们的网络活动。转到 GlassWire 的设置，然后选择 “remote server”用于在本地计算机和远程服务器上安装 GlassWire 后登录到服务器。
GlassWire 在远程服务器上的工作方式与在本地计算机上的工作方式完全相同。查看安全警报、新的应用网络活动、 还有更多！
